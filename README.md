# Solid Earth Literature Monitor

This project is a static web page for presenting curated Solid Earth literature
briefs collected from multiple journals.

The page highlights the latest report, keeps previous reports in chronological
order, and lets readers search articles by title, author, method, region,
journal, and interest keywords.

Usage: Open [litao-t.github.io/article_monitor](litao-t.github.io/article_monitor) directly in a browser.
